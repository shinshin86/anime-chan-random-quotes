# anime-chan-random-quote

This project uses [anime-chan of API](https://github.com/RocktimSaikia/anime-chan)

## Setup

```bash
git clone https://github.com/shinshin86/anime-chan-random-quotes.git
cd anime-chan-random-quote
yarn install
yarn dev
# => localhost:3000
```
